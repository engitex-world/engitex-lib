# Engitex Lib

> Shared TypeScript library used by [engitex-fe](https://github.com/engitex-world/engitex-fe) and [engitex-be](https://github.com/engitex-world/engitex-be)

This library contains common logic, utilities, constants, enums, and types shared between the frontend and backend of the Engitex platform. It is structured to be used as a Git submodule in both projects and supports dependency management, formatting rules, and a clean development workflow.
